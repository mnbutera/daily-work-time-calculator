Daily Work Time Calculator

A simple web-based calculator for tracking work hours throughout the day.

Live Calculator

https://mnbutera.github.io/daily-work-time-calculator/

Features

* Calculate total time worked across multiple work periods
* Automatically deduct lunch or other unpaid breaks
* Five-entry layout for a clean, uncluttered interface
* Runs entirely in your browser—no installation required
* Works on desktop and mobile devices

About

This project began as a personal utility to simplify daily time calculations and eliminate repetitive manual math. It is intentionally lightweight, fast, and easy to use.

Suggestions and improvements are welcome.
